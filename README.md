# JDX Portfolio

Portfolio développé avec Next.js.

Le projet présente le travail de Joey Decroix avec une homepage immersive, une scène 3D interactive, une section projets, ainsi que des pages de détail bilingues FR / EN.

## Commandes de base

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev
```

Créer le build de production :

```bash
npm run build
```

Lancer le projet en production :

```bash
npm run start
```

Lancer le lint :

```bash
npm run lint
```
